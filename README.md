# Adote-Amor-
Adote Amor é um site desenvolvido em HTML5 e CSS3 para incentivar a adoção responsável de cães e gatos. O projeto apresenta informações sobre os benefícios da adoção, catálogo de animais, orientações para adoção e canais de contato da ONG, com layout moderno, intuitivo e responsivo para diferentes dispositivos.
