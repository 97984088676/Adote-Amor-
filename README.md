# Adote Amor 🐾

O **Adote Amor** é um site desenvolvido com **HTML5 e CSS3** com o objetivo de incentivar a adoção responsável de cães e gatos. O projeto apresenta informações sobre os benefícios da adoção, orientações para o processo de adoção, um catálogo de animais disponíveis e os canais de contato da ONG.

O site possui um design moderno, interface intuitiva e layout responsivo, proporcionando uma boa experiência de navegação em computadores, tablets e celulares.

### Tecnologias utilizadas

* HTML5
* CSS3
* Google Fonts (Inter)

### Funcionalidades

* Página inicial com chamada para adoção.
* Seção com vantagens da adoção responsável.
* Catálogo de animais disponíveis.
* Guia com o passo a passo para adoção.
* Área com os canais de contato da ONG.
* Layout responsivo para diferentes tamanhos de tela.

**Objetivo:** conscientizar sobre a importância da adoção responsável e aproximar animais resgatados de famílias que possam oferecer amor, cuidado e um lar seguro.
